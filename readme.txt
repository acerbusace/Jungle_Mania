Author 
------
Alex Patel, Ibrahim Helmy and Kevin Mound

Description
-----------
Original, Designed and Developed Game via Processing. 
The "source_code" folder contains the processing (v2.2.1) version of the game.

Instructions 
------------

1) Download and extract the zip file
2) Launch the executable "Jungle_Maniav2_0.exe" corresponding to your system (32bit or 64bit, in their respective folders).

Note
----
Make sure the directory structure is the same as the one in the zip file. 
"source_code" folder is not needed to run the program.
